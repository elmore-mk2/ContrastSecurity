## Mantis Docker Sample

### Steps
1. Build and Run  
    ```bash
    docker-compose up -d
    ```

2. Access
    ```
    http://localhost:8989/
    ```

3. Configuration  
    - Hostname (for Database Server)  
    mysql
    - Username (for Database)  
    mantisbt
    - Password (for Database)  
    mantisbt
    - Database name (for Database)  
    bugtracker

    click "Install/Upgrade Database" button.

4. Login  
    ```
    administrator/root
    ```
